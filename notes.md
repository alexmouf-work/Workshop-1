# Key Principles of Data Management

## FAIR

Findable
Accessible
Interoperable
Reusable

## Why should I share my data?

The sharing of data is increasingly required by funding bodies such as the Wellcome trust, or the UKRI.

It increases both scientific transparency and trust, as well as enabling reuse of data. 

Crucially, it increases the visibility and citing of your work.

## Best practices for data sharing

- Use community standards such as BIDS, NWB.
- Anonymize sensitive data
- Choose appropriate licenses (eg. CC BY, CC0)
- Include thorough documentation and README files
- Share the data within university/community repositories.

## UCL Research Data Repository (RDR)

https://rdr.ucl.ac.uk
It's a public facing data repo for curated datasets usually after your project is finished/published, based on the FigShare platform. If there is a community-specific repository, use that instead of the UCL one.
Github is more optimal for software.
